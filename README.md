
# Blog App

A simple blogging web application that also servers as a public diary.




## Run Locally

Clone the project

```bash
  git clone https://github.com/riAs-g/Blog-App
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** HTML, CSS

**Server:** Node, Express, EJS
